Steps of installation :

Download the folder Lastfm to have the data.
Download Talaxie Open Studio that will be needed for extract transform and load the data into the database.
Download MySQL Workbench to create the database that will contain the data used for the dashboard.
Download PowerBI to make the dashboard.

On MySQL Workbench, create a database by clicking on the button "Create a new schema in the connected server".
Give a name to the database and download the script create table.sql on the GitHub repository. Open it and copy it into MySQL Workbench. Execute it to create the table.
Install Talaxie and create a project and open the job from the GitHub repository. 
On the tFileList element of the job, select the folder Lastfm to extract the data from this one.
